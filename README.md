## Deploy
1. Put `index.html` (this version) in your GitHub repo (e.g., `ASSESSMENT-SITE`).
2. Add your saved `.quiz` file (e.g., `exam1.quiz`) to the repo root.
3. Enable GitHub Pages → visit the **student link** above.
